## 简介

输入自然语言，通过神经网络抽取时间成分，进行事务安排

任务解析方面运用了句法分析方法，时间解析方面运用了正则表达式匹配



## 运行环境与方法

* 运行环境：ubuntu 18.04   python3

* 训练好的神经网络模型：

  链接: https://pan.baidu.com/s/1qCObi-ItdHEkKle4_2dp3w  密码: cll8

  需要放在Model文件夹中

* 运行指令

  非UI方式启动：

  ```
  python3 Main.py
  ```

  UI方式启动：

  ```
  python3 App.py
  ```
